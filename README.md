# Airbnb-NYC-ml-project
This project explores the Airbnb NYC dataset and builds machine learning models to predict the nightly price of a listing based on features such as room type, neighborhood, availability, and reviews.

## Project Goal
The main question is:
**Can we accurately predict the nightly price of an Airbnb listing in New York City using machine learning methods?**

## Features Used
- neighbourhood_group
- neighbourhood
- room_type
- minimum_nights
- number_of_reviews
- reviews_per_month
- calculated_host_listings_count
- availability_365

## What is included in this repository
- `/notebooks/airbnb_analysis.ipynb`: Full project notebook with data exploration, visualizations, preprocessing, and models.
- `/src/`: Python scripts for preprocessing, visualization, and model training.
- `/results/`: Exported figures and model results.

## Dataset
Dataset used:
**Inside Airbnb NYC (2019)**  
Download link: http://insideairbnb.com/get-the-data/
